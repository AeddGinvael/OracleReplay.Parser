# Oracle

Oracle is a Dota 2 replay parser written in C# ().Net Core 2.2) targeting the Source 2 game engine only.

## Nuget-package
See https://www.nuget.org/packages/OracleReplay.Parser/
or
`Install-Package OracleReplay.Parser -Version 0.1.0`
